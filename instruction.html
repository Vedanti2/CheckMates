🧠 What is the N-Queen Problem?

The N-Queen problem is a classic computer science puzzle that involves placing N queens on an N×N chessboard such that no two queens attack each other.

📏 Rules

  • Only one queen per row and column
  • No two queens can share the same diagonal
  • Valid solutions must place N queens on the board

💡 Algorithm Used: Backtracking

Backtracking is a recursive algorithm that tries to place a queen in a valid position row-by-row. If it reaches a dead-end, it backtracks and tries a different path. 

function solveNQueens(board, row) {
  if (row == N) return true;
  for (let col = 0; col < N; col++) {
    if (isSafe(board, row, col)) {
      board[row][col] = 1;
      if (solveNQueens(board, row + 1)) return true;
      board[row][col] = 0; // backtrack
    }
  }
  return false;
}

⏱️ Time & Space Complexity

Time Complexity: O(N!)
Space Complexity: O(N²) for the board
