# ♟️ CheckMates — N-Queens Solver & Visualizer

CheckMates** is an interactive web-based application that visualizes the **N-Queens problem** using the **backtracking algorithm**.  
It helps users understand algorithmic thinking through step-by-step visual explanations and interactive gameplay.

🌐 Live Demo: https://vedanti2.github.io/CheckMates/

---

 ✨ Features
- ♟️ Interactive chessboard
- 🧠 Step-by-step backtracking visualization
- 👆 Manual queen placement with validation
- 💡 Hints and auto-solve option
- ⏱️ Real-time feedback on valid/invalid moves
- 🎨 Clean UI with smooth animations

---

 🛠️ Tech Stack
- **HTML5**
- **CSS3**
- **JavaScript**
